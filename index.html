🌐 Network Tools

  • Ping — Tests reachability to a host and measures round-trip time.
  • Traceroute — Shows the path packets take to reach a destination. Note: intermediate IPs are not available in the App Store sandbox.
  • DNS Lookup — Queries A, AAAA, MX, CNAME, TXT, NS, and SOA records.
  • DNS Propagation — Checks how a domain resolves across 8 global resolvers simultaneously.
  • Port Scanner — Scans TCP ports on any host.
  • Subnet Calculator — Calculates network ranges from any IP/mask combination.
  • Bonjour Browser — Discovers mDNS/Bonjour services on your local network.
  • Latency Map — Measures TCP latency to major cloud datacenters worldwide.
  • IP Geolocation — Geolocates any IP or looks up your own public IP.
  • Wake-on-LAN — Sends a magic packet to wake a remote machine.

🔒 Security Tools

  • SSL Inspector — Inspects TLS certificate chains and expiry dates.
  • Cert Transparency — Searches certificate transparency logs for any domain.
  • HTTP Headers — Fetches and analyzes HTTP response headers.
  • JWT Decoder — Decodes JSON Web Token header and payload.
  • Password Generator — Generates strong random passwords.
  • Hash Calculator — Computes MD5, SHA-1, SHA-256, SHA-512, and HMAC.
  • TOTP Generator — Generates time-based one-time passwords (RFC 6238).
  • SSH Key Generator — Generates RSA or EC SSH key pairs.
  • Cert Generator — Generates self-signed X.509 certificates for testing.

💻 Terminal & Connections

  • SSH Terminal — Multi-tab SSH with split-pane support. Passwords are stored in the system Keychain.
  • SFTP Browser — Browse and transfer files over SFTP.
  • FTP Client — Connect to FTP servers.
  • RDP — Launches Remote Desktop via Microsoft Remote Desktop.
  • USB Console — Serial console via USB-to-Serial adapters (macOS) or TCP network bridge (iOS/macOS).
  • Connection Manager — Save and manage connection profiles with encrypted export/import.

📋 Known Limitations

  • Traceroute cannot show intermediate router IPs due to App Store sandbox restrictions.
  • Wi-Fi SSID scanning on macOS requires Location Services permission.
  • USB serial console requires a USB-to-Serial adapter on macOS. On iOS/iPadOS, connect via a TCP serial bridge (e.g., Raspberry Pi running ser2net).
  • RDP requires Microsoft Remote Desktop to be installed.
